# Entaru Kernel

Микроядро для псевдо-операционной системы, демонстрирующее базовые принципы работы ядра ОС.

## Описание

Проект представляет собой реализацию ядра для псевдо-операционных систем. Включает базовые компоненты, необходимые для функционирования ОС: управление процессами, памятью, обработку прерываний и системные вызовы.
